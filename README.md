# UGM-Compliance-Framework
Unified Guidance Model (UGM) for AI Safety Compliance - Supplementary Materials

This repository contains the complete supporting materials for the paper "A Unified Guidance Model for Concurrent AI Safety Compliance: Aligning the EU AI Act with ISO 21448 and ISO/PAS 8800."

## Contents:

A. **Safety Sub-Attribute Catalogue** - 60 safety-relevant sub-attributes with definitions and rationales<br>
B. **Requirement Mapping & Rationales** - Complete mapping on requirement level between AIA, ISO 21448, and ISO/PAS 8800 requirements<br>
C. **Integrated Analysis** - Synthesis of framework relationships and interactions<br>
D. **Overlap Relationships** - Direct/Partial/Complementary relationships with detailed rationales<br>
E. **UGM Model Examples** - Full Goal Structuring Notation instantiations for two key sub-attributes<br>



Demonstrates a structured method for concurrent compliance with the EU AI Act alongside domain-specific safety standards (ISO 21448, ISO/PAS 8800) using argument-based assurance cases.
